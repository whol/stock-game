# 股海沉浮 · 历史模拟交易

一个用真实 A 股历史数据玩的模拟交易游戏。从 2024 年 1 月开始，给你 10 万元本金，在历史行情中买卖，体验动态盈亏，解锁成就。

## 玩法

1. **开局**：10 万元本金，时间停在 2024-01-02。
2. **推进时间**：点 ▶ 自动播放（可调速），或 ◀▶ 手动逐日推进。
3. **买卖**：选股票 → 点「买入」/「卖出」，按当天收盘价成交。
   - A 股规则：100 股整数倍、T+1（买入次日才能卖）
   - 费用：佣金万 2.5（最低 5 元）+ 印花税卖出 0.05% + 沪市过户费 0.001%
4. **看盈亏**：顶栏实时显示现金、市值、总资产、总收益率。持仓股连涨连跌会有视觉特效。
5. **解锁成就**：15 个成就，达成后获得称号、奖励资金或解锁新功能。

## 添加更多股票

达成「分散投资」成就（同时持有 ≥3 只股票）后，解锁「添加股票」按钮，支持两种方式：

### 方式一：在线获取（默认）

游戏内输入代码（如 `sz000725` 京东方A），点「获取并导入」，浏览器直连腾讯公开接口拉数据。**多数情况可用**。

### 方式二：粘贴 JSON（兜底）

如果直连失败（CORS / 网络），自动切到粘贴模式：

1. 在 WorkBuddy 主对话执行：
   ```
   westock-data kline sz000725 --start 2024-01-01 --end 2025-06-30 --period day --fq qfq --raw
   ```
2. 复制返回的 JSON 数组
3. 粘贴到游戏文本框，点「解析导入」

## 本地运行

游戏是纯前端，但用了 ES Module + fetch 加载 `data/stocks.json`，需要通过 HTTP 服务器打开（不能直接双击文件）：

```bash
# 进入项目目录
cd 2026-06-30-01-26-51

# 方式一：Python
python -m http.server 8080
# 然后浏览器打开 http://localhost:8080

# 方式二：Node（任选）
npx http-server -p 8080
```

## 数据刷新

游戏历史数据覆盖 2024-01-02 ~ 2025-06-30。若要更新数据：

1. 在 WorkBuddy 主对话执行（替换代码）：
   ```
   westock-data kline sh600519 --start 2024-01-01 --end 2025-12-31 --period day --fq qfq --raw
   ```
2. 把返回 JSON 存到 `data/raw/sh600519.json`
3. 运行 `node scripts/build-stocks.js` 重新生成 `data/stocks.json`

## 技术栈

- HTML + CSS + 原生 JavaScript（ES Module）
- Chart.js v4 + chartjs-chart-financial（K 线图）
- 数据源：腾讯自选股（经 westock-data skill 拉取）
- 存档：localStorage
